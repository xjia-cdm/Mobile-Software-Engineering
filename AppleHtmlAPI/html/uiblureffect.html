Class

UIBlurEffect

An object that applies a blurring effect to the content layered behind a visual effect view.

SDKs

  • iOS 8.0+
  • tvOS 9.0+

Framework

  • UIKit

On This Page

  • Overview
  • Topics
  • Relationships
  • See Also

Overview

Views that you add to the contentView of a visual effect view are not affected by the blur effect.
